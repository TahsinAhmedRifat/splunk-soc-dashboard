# Windows Threat Hunting Dashboard
This project is a SOC Analyst portfolio dashboard created in Splunk.

## Panels Included:
1. Failed Login Attempts
2. Top Hosts with Failed Logins
3. System/Application Errors
4. Top Processes / Events

## Tools Used:
- Splunk 
- Windows Security & Application Logs

## How to Use:
Import the XML file into Splunk to view the dashboard.
